===== Display Server =====

  1. Focus change: damage only borders, not full redraw
     compositor/mod.rs set_focus() calls mark_full_redraw() unconditionally.
     Should compute 4 border-strip rects for old + new focused window and
     call mark_damage instead.

  2. Two-channel blend optimization
     compositor/render.rs blit_to_back_uniform — inactive-dimming does 3
     separate multiplies per pixel. Can fold R+B into one multiply via
     (px & 0x00FF00FF) * alpha trick.

  3. Floating drag minimum on-screen margin
     compositor/mod.rs update_drag() MoveFloating — window can be dragged
     almost entirely off-screen. Clamp to -(w - 64) .. screen_w - 64.

  4. Fullscreen toggle (Super+F)
     Add ShortcutAction::Fullscreen. Set fullscreen_window: Option<WindowId>;
     when set, recalculate_toplevel_layout gives it the full area.

  5. Cursor capture / hiding for games
     Add SetCursorMode { hidden, locked } message. Compositor suppresses
     cursor rendering and accumulates raw deltas for locked window.
